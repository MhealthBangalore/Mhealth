# Mhealth
FRHS (Foundation for Research in Health-Systems) is to develop and apply innovative approaches and models for improving decision making and management practices in health.

FRHS will keep track of all the data such as abortions, live birth, polio etc for each and every places in India. To perform such activities from each and every villages and PHC (Physical Health Center)  FRHS came up in implementing the MHealth (Mobile Health) application.

The main aim of Mhealth is to migrate all the form based data to Sql based database and input the real time data via android applications to a particular database. This data will be used to spool the reports based on the particular keyword and period by using the DHIS
